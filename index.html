Restaurant Tipping Analysis

Visual summary of tipping behavior using the classic tips dataset. Charts are computed in Python (pandas + seaborn + matplotlib)

Quick insights

Tip % Distribution

Overall generosity across all tables.

What this shows

This histogram shows how tip percentage is distributed for all tables, highlighting the most common tipping range and the presence of unusually low or high tips.

Tip % by Day

Comparing tipping patterns across days.

What this shows

Each box summarizes the distribution of tip percentages for a given day, so you can compare median tipping and variability across Thursday, Friday, Saturday, and Sunday.

Weekday vs Weekend

Do people tip differently on weekends?

What this shows

This compares tip percentages on weekdays versus weekends to reveal whether weekend diners are more generous than weekday diners.

Lunch vs Dinner

Meal-time effects on tipping percentage.

What this shows

This chart shows how tipping differs between lunch and dinner, comparing central tendencies and variability for each meal.

Party Size vs Tip %

Are large groups more or less generous?

What this shows

Each point is a table, and the line shows the average tip percentage for each party size, revealing how generosity changes as group size grows.

Tip vs Total Bill

Color encodes tip percentage; point size encodes party size.

What this shows

This scatter plot shows how tip amounts grow with total bill size, while color and point size encode tip percentage and party size, highlighting whether tips scale proportionally with bills.

Average Tip % by Day & Meal

High-level summary combining day of week and meal type.

What this shows

This bar chart aggregates the data to show average tip percentage for each combination of day and meal, summarizing most of the patterns observed in the earlier charts.
